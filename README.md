# SpecFlow_RestSharp
Practicing Specflow and RestSharp 

### Install: 
 - .Net Framework
 - SpecFlow
 - NUnit
 - RestSharp
 - json-server (for the api simulation)
 
 ### Steps (project directory)
  - Start [json-server](https://github.com/typicode/json-server):
    - json-server --watch api_simulator/db.json
  - dotnet test
